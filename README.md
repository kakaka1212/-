# 作品名『温湿度計』 

RASPBERRY Pi 3 をりようした温湿度計を作ってみました！

1分ごとにデータを自動で取得し、設定された値から温度か湿度が外れた場合、警告ランプがつきます。

# デモ画面
 
* 日時、温度、湿度取得
* グラフ表示
* 基盤
 
# 特長、使用技術

この作品はラズパイ,温湿度計(DHT11),LEDを使用しています。

* python
 
# 必要な機器
 
*RASPBERRY Pi 3
* DHT11
